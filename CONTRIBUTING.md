# Contributing to CampusSutra

CampusSutra is developed and maintained by **Varad Lodh**, the original author of this project.

At this stage, external contributions are limited and reviewed directly by the founder/developer.

## Contribution Guidelines

- For suggestions or issues, please open a GitHub Issue in this repository.
- Major changes should be discussed via an issue before creating a pull request.
- All code should follow the existing project structure and coding style.
- Only meaningful and well‑documented contributions will be considered.

## Ownership

The concept, architecture, and core implementation of CampusSutra  
are created and maintained by **Varad Lodh**.
